-- 체크를 하면 게임처럼 포인트 적립이 있어야함.
-- 오늘의 포인트 아래에 매일 갱신되는 오늘의 명언 추가
-- TODO는 시간대가 옵션
-- TODO 에 시간 설정하면 그 시간에 알람
-- 모닝콜 알람 설정 기능,
-- 화면은 월별 화면 필요
-- 일별 화면에서 TODO 관리
-- 구글 로그인 기능 사용, 사용자 데이터는 전부 Firestore 기반으로 관리
-- TODO는 기간 설정 가능
-- TODO 시간은 from to 가 있어서 완료처리를 그 사이에 하면 포인트 적립
-- 획득 포인트는 최대 100p이고 하위 할일이 있으면 넣을 필요가 없고, 하위 할일 포인트의 합산이 포인트로 들어감. 기존에 포인트 설정이 되있더라도 하위 할일이 생기면 하위 할일 포인트 합산으로 변경됨.
-- TODO 추가 / 수정 화면 필요, TODOLIST에서 삭제 필요
-- TODO는 요일 설정 가능하고 설정한게 오늘의 할일 화면에 구별 없이 전부 다 보여짐
-- TODO 하나 안에 세부적인 TODO를 나눠서 추가할 수 있고 (depth는 한개만 더) 하고 그 세부 TODO도 체크박스로 체크 가능
-- 할 일 눌렀을 때 세부 할일 화면 필요함
-- 세부 할일 추가 화면도 필요함
-- 캘린더 화면에 완료한 할 일 개수는 필요 없고, 제일 달성 못한 할일들을 보여줘야함
-- 캘린더 화면 년/월/일 선택이 가능해야함
-- 캘린더 화면에 할일 관리 기능이 필요함, 맨 마지막에 할일 리스트를 보여주고, 추가 및 수정, 삭제가 가능하도록 필요함.
-- 캘린더 화면에서 할일 추가/수정을 할 수 있는데, 같은 화면을 호출하는데 추가일 경우 항목은 다 초기 추가 용으로 값 들어가 있고, 수정일 경우는 수정 눌러서 넘어온 할 일의 데이터 기준으로 초기 입력이 되어 있어야 함.
-- 설정에서 전체적인 앱 색깔 톤을 바꿀 수 있는 커스텀 기능 필요

-- [프로필 구조]
-- 하나의 구글 계정으로 여러 프로필을 만들 수 있고, 앱을 여러 사람이 돌아가면서 사용할 수 있음.
-- 프로필은 구글 계정(ownerUid) 기준으로 여러 개 생성 가능하며, 각 프로필은 독립된 할일/포인트를 가짐.
-- 초기 진입 시 프로필 선택 화면("누가 사용하시나요?")에서 내 프로필 목록이 표시됨.
-- 내 프로필(ownerUid == 내 구글 UID) 선택 → 해당 프로필 기준으로 앱 초기화 (내 할일 화면)
-- 팀원 프로필(다른 ownerUid, 내가 가입한 팀의 멤버 프로필) 선택 → 타인 할일 보기 화면으로 이동
-- 프로필 선택 화면은 두 섹션으로 구성: [내 프로필 목록] + [팀원 프로필 목록]
-- 프로필에 "내 계정 설정" 기능이 있고, 이를 통해 해당 프로필이 구글 계정 소유자(isMyAccount:true)임을 표시
-- isMyAccount=true 프로필: 구글 계정 연동 아이콘/뱃지 표시, 내 계정임을 명확히 구분
-- 첫 로그인 시에 프로필을 못 가져와서 계속 뱅뱅 돌면서 멈춰있는 현상이 있음. 아마도 프로필이 하나도 없어서 그런거 같은데 이 현상 개선 필요. 관련 DB나 소스 수정 필요.
-- 선택된 프로필 기준으로 데이터 읽어와서 앱 초기화
-- 설정에서 프로필 선택과 관리 기능이 필요함
-- 프로필 관리 화면도 필요함
-- 프로필 수정 화면도 필요함
-- 프로필 수정에서 사진 편집하면 팝업으로 남성/여성에 대한 탭이 나눠지고 각 탭별로 30개씩 선택할 수 있도록 해줘야함.
-- 내 계정 외의 다른 프로필(타인 할일 보기)로 들어갔을 때에만 할일에 따봉 기능 추가 가능.
-- 세부 할일에도 따봉 추가 가능, 따봉은 할일 하나당 한번만 설정 가능하며 여러명이 따봉 가능.
-- 따봉 개수가 화면에서 따봉 아이콘에 조그맣게 확인 가능해야 함, 따봉 기능 달 때 코멘트도 같이 달 수 있음.
-- 따봉 아이콘 누르면 누가 따봉 줬는지 확인 가능해야함
-- 따봉 준 사람은 1포인트 획득 가능하며 따봉 받은 사람은 하나당 2포인트 획득 가능.
-- 내 프로필이 아닌 다른 사람 계정의 할일 화면에 대한 별도 화면이 필요함.
-- 다른 사람 할일 화면에서는 따봉 및 댓글을 남길 수 있고, 따봉은 취소가 가능해. 그리고 따봉을 주게 되면 받는 사람의 포인트 총합이 올라가야해.
-- DB 구글 firestore 기반으로 사용자들끼리 공유하며, limit에 걸리지 않기 위한 DB 접근 관리가 필요해.
-- 팀 메뉴를 새로 만들어서 팀 기능을 추가하고, 내가 그 팀에 소속하거나 빠질 수 있게 필요.
-- 팀을 눌렀을 때 팀에 대한 대시보드 화면이 보여짐.
-- 팀 기능 추가해서 팀 내 최고 달성자 확인 가능(주간/월간/년간)
-- 팀에서 따봉 제일 많이 한 사람, 제일 따봉 잘 받은 사람 같은거, 팀 멤버 확인 필요
-- TODO 세부 화면에 URL을 추가할 수 있게 해주고, 링크를 누르면 브라우저가 열리게 해주고, 이 링크는 브라우저에서 공유를 눌러서 받을 껀데, 링크 공유 받았을 때 할일을(세부 할일 포함) 선택해서 그 할일에 링크가 추가되게 해줘.
-- 링크는 여러개를 리스트로 관리할 수 있고, 세부 할 일 화면에서만 보이는게 아니고 일반 할일의 상세 화면에서도 보여야해. 그리고 링크는 수정/삭제가 가능해야해.
-- 프로필 사진을 카카오나 구글 계정의 설정된 사진으로 기본적으로는 바로 가지고 오고, 수정 가능하도록 하자.
-- 오늘의 포인트 글귀 앞에 색깔이 안 차 있는 하트가 있고 할일 완료 체크박스 클릭하면 하트가 나선형 궤적으로 날아가서 그 하트를 색깔을 채우고 포인트 바를 업데이트 해줘야함. 포인트바 올라갈 떄 하트가 한번 깜빡여야함.
-- 오늘의 포인트 하루 전체를 달성할 경우 하트는 색깔이 차고 폭죽이 터지면서 축하.
-- 체크박스 클릭하면 할일 완료 표시 되야 하고 취소 가능하며 취소할 때 포인트가 깎여야함
-- 목표 메뉴에 목표 관리 기능이 필요함.
-- 목표 추가 시 목표를 월별/분기별/년별, 재무/건강/커리어/자격증 등 다양한 카테고리로 선택 가능하고 그게 할일 목록에서 항상 보여야함.
-- 목표는 진척 관리가 필요한 항목과 필요하지 않은 항목이 있음.
